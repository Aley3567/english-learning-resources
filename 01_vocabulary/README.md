# 词汇资料

本文件夹包含CET6核心词汇资料，按字母和主题分类。

## 文件结构

```
01_vocabulary/
├── by_alphabet/          # 按字母分类
│   ├── A_开头.md
│   ├── B_开头.md
│   └── ...
├── by_frequency/         # 按词频分类
│   ├── 高频词汇.md
│   ├── 中频词汇.md
│   └── 低频词汇.md
├── by_topic/            # 按主题分类
│   ├── 学术词汇.md
│   ├── 商务词汇.md
│   └── 日常生活词汇.md
└── word_lists/          # 词汇表汇总
    ├── 核心词汇3000.md
    └── 重点词汇1000.md
```

## 学习建议

1. 每天学习1-2个字母的词汇
2. 结合例句记忆单词用法
3. 定期复习已学词汇
4. 使用间隔重复法加强记忆
