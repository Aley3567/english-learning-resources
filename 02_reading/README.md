# 阅读材料

本文件夹包含CET6阅读训练材料，包括真题、模拟题和专项练习。

## 文件结构

```

│   ├── 2023年真题.md

│   └── 历年真题汇总.md
├── practice/             # 模拟练习
│   ├── 模拟题1.md
│   ├── 模拟题2.md
│   └── 模拟题3.md
├── skills/               # 阅读技巧

│   ├── 词义推测技巧.md

├── vocabulary_context/   # 上下文词汇
│   ├── 学术文章词汇.md
│   ├── 新闻报道词汇.md
│   └── 科技文章词汇.md
└── analysis/             # 阅读分析
    ├── 答案解析.md
    └── 错题总结.md
```

## 训练建议

1. 每天完成1-2篇阅读练习
2. 总结常见题型和解题技巧
3. 积累阅读中的高频词汇
4. 分析错题原因，改进阅读策略
